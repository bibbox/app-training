eb3kitlearning

Welcome! eb3kitlearning is the training component of the eb3kit. It includes self-learning resources (videos, tutorials...) and training toolkit. It comes with a minimal set of resources and can be fed with your own material.

The eb3kitlearning is based on WordPress, with the theme Hestia and several plugins. It has been tested with WordPress 4.9.3

Plugins details

  1. Elementor (mandatory) is an advanced frontend drag & drop page builder.
  2. iframe (mandatory) shortcode is the replacement of the iframe html tag, which is removed by WordPress because of security reason, and accepts the same params as iframe html tag does.
  3. LearnPress (mandatory) is a WordPress complete solution for creating a Learning Management System (LMS). It can help you to create courses, lessons and quizzes.
  4. Orbit Fox Companion (mandatory) extends your theme functionality with various modules like Social Media Share Buttons & Icons, custom menu-icons, one click import page templates, page builder addons and free stock featured images.
  5. The Events Calendar (optional) is a carefully crafted, extensible plugin that lets you easily share your events. It is not a mandatory plugin, you can replace it by your favorite calendar plugin.
  6. WP Statistics (optional) tracks statistics for the site.  You can replace it by your favorite statistics plugin.
